
codezero - visual web scraping script builder

Technologies used :

Java 1.8
Spring 4
Hibernate 4
MySQL 5
Angularjs
Jquery
Bootstrap css
Selenium/chrome driver
mongodb

Setup :

git clone https://github.com/renga99/codezero.git
cd codezero
mvn tomcat:run

Access on:

http://localhost:8080/codezero/
