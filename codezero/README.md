
codezero - visual web scraping script builder

Technologies used :

Java 1.8<br />
Spring 4<br />
Hibernate 4<br />
MySQL 5<br />
Angularjs<br />
Jquery<br />
Bootstrap css<br />
Selenium/chrome driver<br />
mongodb

Setup :

git clone https://github.com/renga99/codezero.git<br />

cd codezero<br />

mvn tomcat:run<br />

Access on:

http://localhost:8080/codezero/
